# Zandbox — Idea Constellation

**Learn. Scale. Disrupt.**

An interactive sandbox for organizing and exploring project ideas across AGI research, federal contracting, rural infrastructure modernization, and zero-budget AI systems.

---

## Overview

Zandbox serves as a central hub for tracking interconnected research initiatives and business development projects for **Visionblox LLC** and **Zuup Innovation Lab**. The interface visualizes ideas as an evolving constellation—each node representing a concept, platform, or methodology under active development.

## Domains

| Domain | Focus |
|--------|-------|
| **AGI/ASI Research** | Recursive Cognitive Architecture (RCA), CHC theory integration, superintelligence pathways |
| **Federal Capture** | OTA mechanisms, SOSSEC consortium, GSA MAS, capture methodology |
| **MVCI Systems** | 12-agent zero-budget infrastructure using n8n, Ollama, SQLite |
| **Rural Infrastructure** | Critical Access Hospitals, Rural Water Associations, digital transformation |
| **Digital Transformation** | Tier 2/3 supplier economics, platform ROI analysis |
| **Zuup Platforms** | Aureon, Veyra, Civium, PodX, Symbion, QAWM |

## Key Projects

### Recursive Cognitive Architecture (RCA)
Eight-layer framework for building self-aware, recursively improving AI systems grounded in Cattell-Horn-Carroll cognitive theory.

### 12-Agent MVCI System
Zero-budget corporate infrastructure using open-source tools:
- **Stack:** n8n + Ollama (Mistral-7B-Instruct) + SQLite
- **Agents:** Executive Coordinator, Decision Logger, Contract Intake, Vendor Due Diligence, Spend Control, and 7 more
- **Target:** 80% TCO reduction with mandatory approval gates

### Autonomous Crypto Agents
Research exploration into AI agents with crypto wallets for autonomous wealth generation and swarm coordination.

## Usage

### Local Preview
```bash

# Open in browser
```

## Technical Stack

The interface is a single self-contained HTML file with:
- No build tools required
- No external JavaScript dependencies
- Google Fonts (Playfair Display, JetBrains Mono)
- CSS custom properties for theming
- Responsive design (mobile-friendly)
